1. How to Use Bootstrap Utilities

Bootstrap utilities are just CSS class names you write inside the class="" attribute of any HTML element. No setup, no extra config â€” just link Bootstrap and start adding classes.

Step 1: Link Bootstrap CSS in your <head>

You need to load Bootstrap's CSS before you can use any utility class. Paste this <link> inside your <head> tag.

<head>
  <!-- Required meta tags -->
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>My Page</title>

  <!-- âœ… Bootstrap CSS CDN -->
  <link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.2/dist/css/bootstrap.min.css"
        rel="stylesheet">
</head>

Step 2: Write utility classes inside class=""

Add Bootstrap class names inside the class attribute of any HTML element â€” separated by spaces. Stack as many as you need.

<!-- Single utility class -->
<div class="mt-3">I have a top margin of 1rem</div>

<!-- Multiple utilities stacked (separated by spaces) -->
<div class="container mb-4 p-4 bg-primary text-white rounded">
  This box has: container width, bottom margin,
  padding, blue background, white text, rounded corners.
</div>

<!-- Works on ANY HTML element -->
<p      class="text-center fw-bold text-danger">A paragraph</p>
<span   class="bg-warning text-dark px-2 rounded">A span</span>
<button class="btn btn-success mt-2 w-100">A button</button>
<img    class="rounded-circle border border-primary" src="photo.jpg">

Step 3: Understand the class naming pattern

Bootstrap utility names follow a consistent formula: {property}{side}-{size} or {property}-{value}. Once you know this, you can guess most classes without looking them up.

bg-primary
bg = background
primary = value
mt-3
m = margin
t = top (side)
3 = 1rem (size)
text-center
text = text-align
center = value
d-flex
d = display
flex = value
col-md-6
col = column
md = breakpoint (â‰¥768px)
6 = 6 of 12 columns
px-4
p = padding
x = left + right
4 = 1.5rem (size)

Step 4: Add Bootstrap JS optional

Utility classes only need CSS. But for interactive components like dropdowns, modals, and collapsible navbars, add this before your closing </body>. 

  <!-- âœ… Bootstrap JS Bundle (includes Popper.js) -->
  <script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.2/dist/js/bootstrap.bundle.min.js"></script>
</body>

Full Starter Boilerplate

Copy this and you're ready to start writing Bootstrap utilities immediately. 

<!DOCTYPE html>
<html lang="en" data-bs-theme="dark">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>My Bootstrap Site</title>

  <!-- Bootstrap CSS -->
  <link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.2/dist/css/bootstrap.min.css"
        rel="stylesheet">

  <!-- Your custom CSS (Optional) -->
  <link rel="stylesheet" href="style.css">
</head>
<body>

  <!-- Start using Bootstrap utilities here! -->
  <div class="container mt-5">
    <h1 class="text-center fw-bold text-primary">Hello, Bootstrap!</h1>
    <p class="lead text-center text-muted mt-3">Start building here.</p>
  </div>

  <!-- Bootstrap JS Bundle -->
  <script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.2/dist/js/bootstrap.bundle.min.js"></script>
</body>
</html>

2. Spacing Utilities (Margin & Padding)

Format: {property}{sides}-{size}

Properties:

  • m - margin
  • p - padding

Sides:

  • t - top
  • b - bottom
  • s - start (left in LTR)
  • e - end (right in LTR)
  • x - both left and right
  • y - both top and bottom
  • blank - all sides

Sizes:

  • 0 - 0
  • 1 - 0.25rem (4px)
  • 2 - 0.5rem (8px)
  • 3 - 1rem (16px)
  • 4 - 1.5rem (24px)
  • 5 - 3rem (48px)
  • auto - auto

Examples:

mt-3 - Margin top 1rem
p-4 - Padding all sides 1.5rem
mx-auto - Margin left and right auto (centers element)

  • m-{0-5} - margin
  • p-{0-5} - padding

  • bg-opacity-{25-100}

  • fs-{1-6} - font size
  • fw-{bold,normal,light}

  • d-flex - flex container
